ReportingDashboard
  Overview:
  The ReportingDashboard is a lightweight web-based application for visualizing and analyzing movie rental data. This project provides a simple interface to query and display data from multiple data tables related to a movie rental system, such as customers, memberships, payments, and more. The dashboard is designed to be intuitive and interactive, making it ideal for monitoring key data points.

  Steps to run:
  1) Clone
    git clone https://github.com/ryannlle/ReportingDashboard.git
    cd ReportingDashboard
  2) Install flask (if not installed)
    pip install flask
  3) Run the application
    python app.py
